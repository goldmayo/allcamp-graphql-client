## 설치 방법

※[https://github.com/goldmayo/allcamp-graphql-service](https://github.com/goldmayo/allcamp-graphql-service)를 먼저 설치 후 실행 시켜 주세요.

```sh
> git clone https://github.com/goldmayo/allcamp-graphql-client.git
> npm install
> npm run build
> npm run start
# or
> npm run dev
> npm run storybook
```
